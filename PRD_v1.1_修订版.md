# Snapdata 数达 - 产品需求文档（React 实现版）

**版本**：v1.5（React Runtime）  
**更新日期**：2026-02-22

---

## 1. 产品定位

Snapdata 是一个浏览器端 Excel 分析工具，面向非技术用户，提供上传、筛选、主分析、交叉分析、智能结论、复制与导出能力。

---

## 2. 技术栈

- Vite
- React + TypeScript
- Tailwind CSS
- zustand（状态管理）
- xlsx（Excel 解析与导出）

---

## 3. 当前功能范围（已实现）

### 3.1 上传与解析
- 支持 `.xlsx/.xls`
- 点击上传 + 拖拽上传
- 默认读取第一个工作表
- 解析后展示文件名、大小、行数、列数、时间
- 表头自动清理 HTML 标签（如 `<span>`）仅保留纯文本

### 3.2 筛选
- 多条件 AND 筛选
- 操作符：`=`、`!=`、`>`、`>=`、`<`、`<=`、`contains`、`not_contains`
- 支持应用筛选和清空筛选

### 3.3 分析工作区（多任务）
- 支持新增/删除多个分析题目（Task）
- 每个 Task 独立保存：
  - 主分析列
  - 百分比/频数模式
  - 排序方式
  - 小数位
  - 拖拽排序顺序

### 3.4 主分析
- 多选值按英文逗号 `,` 分隔并去重计数
- 基数行为 `Base (n)`（主列非空样本）
- 支持均值行（选项为数值型时）
- 支持拖拽行排序，并联动当前 Task 的交叉分析行顺序
- 支持复制主分析表

### 3.5 交叉分析（每 Task 多块）
- 每个 Task 支持添加多个交叉分析块
- 交叉块可独立配置交叉列
- 交叉块删除按钮为圆形 `×`，位置与交叉表右边缘对齐
- 支持 TopN 差异高亮（1/2/3）
- 支持灯泡显著性（Z 检验，阈值 `|Z| > 1.44`）
- 支持智能结论生成与复制
- 支持复制交叉表
- 表格视觉增强：冻结前两列、条形背景、均值行（数值选项）

### 3.6 批量导出
- 支持一键导出“全部题目 + 全部交叉块”到同一 Excel 文件
- 导出为多 Sheet 结构（按题目和交叉块拆分）

---

## 4. 状态结构（核心）

```ts
{
  dataset: ParsedDataset | null,
  filters: FilterCondition[],
  filteredRows: DataRow[] | null,
  isParsing: boolean,
  parseError: string | null
}
```

分析任务和交叉块状态在工作区组件内维护（React state），与筛选后数据联动。

---

## 5. 统计口径（当前实现）

- 主分析分母：分析列非空样本数
- 交叉列基数：主列有效且命中该交叉值的样本数
- 多选拆分：仅按英文逗号 `,`
- 灯泡规则：每行最高列值 vs 次高列值做两比例 Z 检验

---

## 6. 交付说明

- 开发入口：`npm run dev`
- 发布构建：`npm run build`
- 离线单文件构建：`npm run build:offline`（生成项目根目录 `index.offline.html`）
- 当前页面为 React-only 运行，不依赖 legacy iframe
