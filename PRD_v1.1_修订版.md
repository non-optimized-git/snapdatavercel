# Snapdata 数达 - 产品需求文档（实现对齐版）

**版本**：v1.3（As-Is）  
**更新日期**：2026-02-21  
**文档目的**：与当前代码实现保持一致，作为后续迭代基线。

---

## 1. 产品概述

### 1.1 产品定位
Snapdata 数达是一款浏览器端 Excel 数据分析工具，面向非技术用户，支持上传、筛选、频率分析、交叉分析、差异标记、智能结论、复制和导出。

### 1.2 当前视觉基线
- 固定视觉方案：`B Atlas`
- 不提供主题切换入口
- 不提供 UI 实验室配置入口

---

## 2. 当前实现范围（已上线能力）

### 2.1 数据上传
- 支持格式：`.xlsx`、`.xls`
- 上传方式：点击上传 + 拖拽上传
- 处理方式：前端本地解析（SheetJS）
- 解析范围：默认读取第一个工作表
- 上传反馈：进度条 + 文件大小 + 行列统计

### 2.2 数据筛选
- 支持多条件 AND 过滤
- 操作符：`=`、`!=`、`>`、`>=`、`<`、`<=`、`contains`、`not_contains`
- 数值列支持 datalist 候选值，文本列支持下拉 + 自定义输入
- 应用筛选后自动刷新全部分析模块

### 2.3 主分析
- 每个任务块可选择一个分析列
- 统计逻辑：按逗号拆分后去重计数（用于多选题）
- 显示模式：百分比 / 频数切换
- 排序：升序 / 降序
- 精度：小数位 0~4
- 拖拽：可拖动重排主分析行（Sortable）
- 可复制主分析表格
- 可计算 Mean 行（当选项可解析为数字）

### 2.4 交叉分析
- 每个主分析任务可添加多个交叉块
- 交叉列同样按逗号拆分
- 支持 TopN 差异高亮（1/2/3）
- 支持灯泡显著性标记（Z 检验）
- 可复制交叉表格

### 2.5 智能结论
- 基于灯泡高亮点自动生成结论文本
- 可一键复制结论文本

### 2.6 导出
- 导出入口：下载以上所有表格
- 生成单个 Excel 文件（1 个 Sheet：`分析汇总`）
- 当前导出内容：主分析表（按任务汇总）
- 交叉表/结论未写入导出文件

### 2.7 复制能力
- 优先使用 Clipboard API
- 回退到 `execCommand('copy')`
- 再回退到 `writeText`

### 2.8 在线状态与 PWA
- 支持在线/离线状态显示
- 页面引用 `manifest.json`
- 注册 `sw.js`

### 2.9 依赖加载（新增）
- 启用 `CDN 优先 + 本地 vendor 兜底`
- 当前运行时依赖：
  - `SheetJS` (`XLSX`)
  - `Sortable.js`
- 加载顺序：
  1. 先加载 CDN
  2. CDN 失败时加载本地 `vendor` 目录
  3. 若均失败，页面显示依赖加载错误提示

## 4. 当前统计与算法口径（以代码为准）

### 4.1 主分析统计
- 每行：若单元格非空，按 `,` 拆分并去重后计数
- 分母：主列有效样本数（非空行）

### 4.2 交叉表统计
- `mainCol` 与 `crossCol` 均按 `,` 拆分
- 交叉列基数：含主列有效值且命中该交叉值的行数
- 交叉单元格：命中对应主选项与交叉值的计数/占比

### 4.3 显著性（灯泡）
- 方法：两比例 Z 检验
- 阈值：`|Z| > 1.44`（85%置信）
- 比较方式：每行“最高值”与“第二高值”比较

## 7. 已知限制（当前未实现）

1. 不支持 `.csv` 上传。
2. 仅解析首个工作表，不能切换多 sheet。
3. 交叉显著性采用“最高 vs 第二高”口径，不是“分组 vs Total”口径。
4. 导出仅包含主分析汇总，不包含交叉表与智能结论。
5. 多选分隔符固定为英文逗号 `,`，未支持 `，`、`、`、`;`、`|` 的列级配置。
6. 仍有部分内联事件绑定（`onclick`），后续维护成本较高。
7. 字体仍依赖 Google Fonts 在线加载（失败时回退系统字体）。

## 8. 下一版本（v1.4）建议方向

1. 支持 CSV 与多工作表选择。
2. 统一多选分隔符策略（列级可配置）。
3. 显著性口径切换（最高vs次高 / 分组vsTotal）。
4. 导出扩展为多 Sheet（主分析、交叉分析、结论、筛选条件）。
5. 逐步移除内联事件，转向模块化事件绑定。
6. 字体本地化（可选）并纳入 vendor 兜底链路。
