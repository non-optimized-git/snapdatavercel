# Snapdata 演示视频包（90 秒）

## 1) 你可以直接用的素材

- 演示数据文件：`/Users/yuanyi.li/Documents/New project/demo-data.xlsx`
- 启动命令：

```bash
cd "/Users/yuanyi.li/Documents/New project"
npm run dev
```

- 演示地址：`http://127.0.0.1:5173/`

---

## 2) 90 秒逐秒脚本（镜头 + 口播）

### 0s - 8s：开场与定位
- 镜头：停留首页，标题与上传区同框。
- 口播：
  `这是 Snapdata，一款给非技术用户用的浏览器端 Excel 分析工具。`

### 8s - 18s：上传数据
- 镜头：点击上传 `demo-data.xlsx`，等待文件摘要出现。
- 口播：
  `我上传一份调研数据，系统会自动解析首个工作表，并显示行列信息。`

### 18s - 32s：数据筛选
- 镜头：
  1. 点击 `+ 添加筛选条件`
  2. 条件1：`城市 = 深圳`
  3. 条件2：`年龄段 = 25-34`
  4. 点击 `应用筛选`
- 口播：
  `这里支持多条件 AND 筛选。比如我只看深圳、25 到 34 岁人群。`

### 32s - 50s：主分析
- 镜头：
  1. 点击 `添加分析题目`
  2. `选择分析列...` 选 `购买意向`
  3. 在 `百分比 / 频数` 间切换一次
  4. 点击 `▲` 或 `▼` 演示排序
- 口播：
  `每个分析题目都能独立设置口径。可以在百分比和频数间切换，也能快速排序。`

### 50s - 72s：交叉分析 + 差异/显著性
- 镜头：
  1. 点击 `+ 添加交叉分析`
  2. `选择交叉列...` 选 `性别`
  3. 点击 `差异`，再点 `1`/`2`（TopN 高亮）
  4. 点击灯泡图标（显著性）
  5. 向下轻滚动展示 `智能结论`
- 口播：
  `交叉分析可以一键查看分组差异。TopN 高亮和显著性灯泡能快速定位真正有意义的差别，并自动生成智能结论。`

### 72s - 84s：复制能力
- 镜头：依次点击主分析表 `复制`、交叉表 `复制`、智能结论 `复制`。
- 口播：
  `分析结果支持一键复制，直接粘贴到报告、邮件或 IM。`

### 84s - 90s：批量导出收尾
- 镜头：点击底部 `下载以上所有表格`。
- 口播：
  `最后，一键导出全部题目和交叉块到同一个 Excel 文件，完成从数据到结论的闭环。`

---

## 3) 录制参数（推荐）

- 分辨率：`1920x1080`
- 帧率：`30fps`
- 比例：`16:9`
- 录制区域：仅浏览器窗口（避免桌面干扰）
- 鼠标：保留鼠标指针与点击高亮
- 单条视频时长：`80-100 秒`

---

## 4) 一次过拍摄动作清单（照着点）

1. 启动项目并打开 `http://127.0.0.1:5173/`
2. 开始录屏（macOS：`Command + Shift + 5`）
3. 上传 `demo-data.xlsx`
4. 做两条筛选并 `应用筛选`
5. 添加分析题目并选择 `购买意向`
6. 切 `百分比/频数`，点一次排序
7. 添加交叉分析并选择 `性别`
8. 开 `差异`，再开灯泡
9. 展示智能结论与三个复制按钮
10. 点 `下载以上所有表格`
11. 停止录屏

---

## 5) 备选口播（更短版，30 秒）

`Snapdata 能在浏览器里完成 Excel 的上传、筛选、主分析和交叉分析。我们可以快速查看 TopN 差异和显著性，并自动得到智能结论。结果支持一键复制，也支持把所有题目批量导出到同一个 Excel，适合日常调研和业务复盘。`
